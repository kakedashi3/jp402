# jp402 ブランド / ロゴ

コンセプト: **「402 という発見の合図に、みんなが集まる（収斂）」**。
`jp4◎2` ワードマークの `0` を ◎（同心リング＋中心の芯）に置換。◎ は **スポーク無し**（＝脱・歯車。重工業/モビリティ感を排除）で、レーダーリップル＋中心点＝ディスカバリー層の収斂を表す。

## テーマ衝突を避ける鉄則

サイトの主役色が紫（`#6c5ce7`）かつ hero が紫グラデのため、**「紫＋黄」固定のロゴは置き場所で必ず消える**。背景で反転させる:

- **黄 `#ffe27a` は暗い面（紫 hero）でだけ使うスパーク**。明るい面では黄を使わず紫ワントーンにする（白地で黄は沈む）。
- 紫 hero の上に紫ロゴ＝同化。→ hero では必ず **白（反転）版**を使う。

## ジオメトリ（再現用）

- ワードマーク: viewBox `0 0 180 64` / font-size 40 / weight 800 / baseline y=44。
  `jp4`=text-anchor:end x92、`2`=text-anchor:start x132（◎中心 x112,y29 を挟む＝フォント幅に依存せず整列）。
- ◎: 外環 r14(sw4) + 内環 r7.5(sw2.5) + 芯 r3.5。favicon は r8.5(sw3)+芯 r3 に簡略。

## 既知の制約 / 次の詰め

- **ワードマークの `jp4`/`2` は `<text>`（サイトのフォントスタック）**。サイト上は正しく出るが、**印刷・配布用に完全再現するなら vector エディタでアウトライン化**（フォント非依存パス化）すること。ここでは生成不可。
- 整列は `text-anchor` で ◎ を中心に固定しているのでフォント幅変動に強いが、極端に字幅の違うフォントでは左右余白が変わる。bbox を厳密に詰めたい場合はアウトライン化後にトリミング。
- ◎ の同心リング数・線幅・楕円率は調整余地あり（収斂感 vs 「0」可読性のトレードオフ）。
